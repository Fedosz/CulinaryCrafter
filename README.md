# CulinaryCrafter - программный проект 3 курса НИУ ВШЭ ФКН ПИ

## 1. Введение
### 1.1. Наименование программы
* Наименование программы на русском языке: «Кулинарный Мастер».  
* Наименование программы на английском языке: “CulinaryCrafter”.  

### 1.2. Краткая характеристика области применения программы
Данная программа разработана с целью обеспечения удовлетворения потребностей широкого круга пользователей, включая гурманов и кулинаров всех уровней мастерства. В современном мире еда играет важную роль в повседневной жизни людей, являясь неотъемлемой частью культуры и общения. Многие люди постоянно ищут новые рецепты и кулинарные идеи, используя различные онлайн-платформы, такие как YouTube, и приложения для сохранения рецептов. Однако часто возникают проблемы с организацией сохранённых рецептов и их последующим поиском. Наше приложение призвано решить эти проблемы, предоставляя удобный инструмент для поиска, хранения и организации рецептов.  

## 2. Основания для разработки
### 2.1. Документы, на основании которых ведётся разработка
Программа выполняется в рамках темы курсовой работы в соответствии с учебным планом подготовки бакалавров по направлению 09.03.04 «Программная инженерия» Национального исследовательского университета «Высшая школа экономики», факультет компьютерных наук, департамент программной инженерии.

# Заключение

Проект «CulinaryCrafter» представляет собой удобное приложение для поиска и хранения кулинарных рецептов, ограничивающее проблему разрозненности источников и сложностей организации информации. В рамках разработки были определены цели, реализована основная функциональность и проведено тестирование ключевых модулей. Полученный результат демонстрирует практическое применение знаний по программной инженерии и создаёт основу для дальнейшего развития приложения.
